Some Rules of QuizySearch
1. You will have only 90 seconds to solve each Question

2. Each Question consists of two parts :
Solving the MCQ Quiz Question and Search for the Question's Answer inside the word box

3. Once you select your answer, it can't be undone.

4. For quiz : You will get +1 marks if you choose the correct answer, else 0 marks if wrong answer (No Negative Marking)

5. If you choose wrong answer, correct answer will automatically be Displayed.

6.You can Search for the correct answer in the word Box but you won't get any marks for it.

7. Letters in Search Box can overlap and words can appear vertically, horizontally, or diagonally - backwards too!

8. Once the Time is up, the Next Question is Displayed automatically

9. You can't exit from the Quiz while you're playing.

10. You'll get points only on the basis of your correct answers in the MCQ's
